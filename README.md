# project Youtic (UpWork ID: 16667115)

## Installation

Unzip the ZIP file.

## Configuration

Edit the file `settings.ini`. It should contains the following information:
```ini

[bing]
api_key = my-bing-api-key
results_limit = 50
market = en-US
# latitude = 47.603450
# longitude = -122.329696

[diffbot]
token = my-diffbot-token

```
